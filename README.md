Clone esp-idf
============= 
- mkdir -p $HOME/git
- git -C $HOME/git clone https://github.com/espressif/esp-idf.git


Setup install:
- cd ~/esp/esp-idf
- ./install.sh
- source export.sh in esp-idf dir

Setup eclipse:
- Install expressif eclipse plugin: https://github.com/espressif/idf-eclipse-plugin/blob/master/README.md
